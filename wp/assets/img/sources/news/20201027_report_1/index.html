【動画公開】現場が求める監視カメラソリューション
オンライン説明会

アムニモは、あらゆる産業用システムのエッジコンピューティングに適した機器「Edge Gateway amnimo Gシリーズ AG10」関するオンライン説明会を開催し、そのプレゼンテーションを動画にて公開開始しました。

●「Edge Gateway amnimo GシリーズAG10」発表内容の詳細はこちらから


産業におけるIoTの普及が進むにつれて、利用者の業務プロセスの改善に成功した例が増えてきていますが、同時に「厳しい外部環境下で障害を抑制したい」、「仮に障害が発生した場合でも現場に駆けつけることなくリモートから対処したい」という運用上の課題が見えてきています。

また監視カメラなどの用途では、映像のような大容量のデータをエッジで高度に処理したいという需要も現れてきました。

今回アムニモが発表した「Edge Gateway amnimo GシリーズAG10」は、このようなエッジコンピューティングに対する期待に応える形で開発されました。

【説明会でのプレゼンテーション概要】

アムニモ CDO（Chief Development Officer）の中林千晴は、「画像や動画をIoTで活用したいニーズは多いが、コストや遅延、セキュリティ、運用面での課題などを抱え、なかなかうまくいっていないのが現実。アムニモのEdge Gatewayは、これらの課題を解決するものである。今回の新製品を皮切りに、屋外モデルやIoTルータ、AIモデルなどにシリーズ化し、アプリケーションを拡充していく」と発言しました。

また、アムニモ IoTエッジビジネス事業部 事業部長の小嶋修は、「監視カメラソリューションでは、記録するSDカード録画が停止したり、電源ノイズでカメラが突然動かなくなったり、遠隔での映像管理ができなかったり、保守運用においてさまざまな課題があった。従来はこれらを人手で対応していたが、アムニモのEdge Gatewayを組み合わせることで、これらを抜本的に効率化できる」と導入メリットについて発言しました。
