「IIFES2019」出展レポート

2019年11月25日(水)～27日(金)、アムニモ株式会社がオートメーションと計測の先端総合技術展「IIFES 2019」に出展しました。産業向けIoTを、より手軽に安く利用するためのアムニモ社のサービスの仕組みと、具体的な事例を紹介いたしました。

＜展示会概要＞

イベント名：IIFES 2019
開催地：東京ビッグサイト西ホール1Fアトリウム
開催期間：2019年11月25日(木)～27日(金)
総来場者数：50,252名
アムニモ出展規模：1コマ（中堅・中小製造業IoTパーク内）
アムニモ展示内容：実演デモ3種

ブース展示

ブースに「amnimo sense（アムニモセンス）」を活用した4つのレシピと、応用サービスである「pump guard（ポンプガード）」を展示しました。

  1. 温度監視レシピ
  2. 設備稼働監視レシピ
  3. 工場向け部品在庫量見える化レシピ
  4. pump guard（うずまきポンプの異常動作を遠隔監視。重大な故障を未然に防ぐ）
