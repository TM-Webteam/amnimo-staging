「SORACOMパートナースペース 認定済デバイスパートナー」に認定および『IoTルーター amnimo Rシリーズ AR10』が「SORACOM認定デバイス」として登録

この度アムニモは、株式会社ソラコムのパートナープログラム「SORACOMパートナースペース※1（以下SPS）」において「SPS認定済デバイスパートナー※2」として認定されました。

また、アムニモのIoT用LTEルーター「IoTルーター amnimo Rシリーズ AR10」がSORACOM AirやSORACOM　Beamに対応し、「SORACOM認定デバイス」として登録されました。これにより、すでに認定デバイスに登録されているLinux OS搭載のLTEゲートウェイ「Edge Gateway amnimo Gシリーズ AG10」　と共に、お客様に安価な通信料金でのIoTサービスのご提供が可能となります。

詳しくは以下をご覧ください：
ソラコム プレスリリース
「SORACOMパートナースペースに新たに5社の認定済パートナーが参画」
https://soracom.com/ja-jp/news/20210623-6/

※1 SORACOMパートナースペースとは、SORACOMのパートナープログラムです。 IoTシステムをご検討のお客さまは、実績と知見を持つSPSパートナーのサポートを得ることで、迅速にIoT活用をはじめられます。

※2 SPS 認定済デバイスパートナーとは、 SORACOM Air等、株式会社ソラコムの回線サービスに対応したセンシングデバイス、ゲートウェイ、LTE/3G 対応モジュールなどの IoT デバイスを提供する企業です。

＜SORACOM認定デバイス＞

■Linux OS搭載のLTEゲートウェイ

『Edge Gateway amnimo Gシリーズ AG10』
厳しい環境下でも安定的に動作し、障害時もリモートで対応できます。５つのEthernetポートを実装されており、そのうち４つのポートではPoEにより監視カメラに電力を供給。


■高信頼性で小型軽量なLTEルーター

『IoTルーター amnimo Rシリーズ AR10』
お客様のIoT機器とEthernetによって接続され、IoT機器からのインターネット通信を可能とする汎用性の高いルーターです。２つのEthernetポートを実装した産業用LTEルーターであり、Ethernetのポートを経由して、さまざまな産業用途のIoT機器と接続することが可能です。また、日本におけるISDNサービスの提供終了や世界各地で進行する3Gネットワークの廃止に伴う既存の3G端末の置き換えにも適した通信機です。



【株式会社ソラコム様からのエンドースメント】

アムニモ様のSPS認定済デバイスパートナーへの参画を心より歓迎します。SORACOMは、IoT通信とシステム構築に必要なプラットフォームサービスを提供するIoTプラットフォームです。アムニモ様の産業用IoTデバイスにおける知見が活かされたIoTルーター/ゲートウェイと、SORACOMの組み合わせが、製造業やインフラ業などの現場のデジタル化を力強くサポートするものと確信しています。
