CEマーク取得のお知らせ

この度、アムニモの製品である「Edge Gateway Indoor amnimo GシリーズAG10」は、CE（Conformité Européenne）マークを取得しましたことをお知らせします。

【CEマークについて】
正式名称は、Conformité Européenne。フランス語で「欧州の法律に適合している」という意味。製品をEU加盟国へ輸出する際に、安全基準条件（使用者・消費者の健康と安全及び共通利益の確保を守るための条件）を満たすことを証明するマーク。

これら認証の取得により、「Edge Gateway Indoor amnimo GシリーズAG10」を今後、EU各国へ輸出することが可能となりました。

今後ともアムニモは、お客様に安全で安心な製品・サービスをお届けします。
引き続きご愛顧のほど何卒宜しくお願い致します。
