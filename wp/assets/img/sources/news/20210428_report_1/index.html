エッジゲートウェイシリーズ新機能の提供開始のお知らせ

平素は弊社サービスをご利用いただきまして、誠にありがとうございます。
この度4月28日より、LTE通信デバイス（Edge Gateway）とクラウドサービス（デバイス管理システム）において、下記の機能がご利用いただけるようになりましたのでご案内いたします。

記

１．「Edge Gateway amnimo Gシリーズ AG10」におけるGUIによる端末の設定機能

概要：これまではAG10の設定を行なう際にはコマンドラインインターフェース(CLI)にアクセスし、コマンドの操作により設定を行なっていただく必要がありましたが、本機能によりWebブラウザ上で、画面を見ながら簡単に設定を行なうことが可能になりました。

２．「デバイス管理システム」向け自動初期設定機能

概要：自動初期設定では、あらかじめクラウド側に通信デバイスに適用する設定ファイルを指定しておけば、通信デバイスがインターネット経由で「デバイス管理システム」にアクセスして自動的に自デバイスの設定を行なえます。これにより通信デバイスの初期設定にかかる手間を減らすことができます。
適用対象となる機種：「Edge Gateway amnimo Gシリーズ AG10」、「IoTルーター amnimo Rシリーズ AR10」

現在「Edge Gateway amnimo Gシリーズ AG10」のサンプル機をご利用いただいているお客さまは、本機能をご利用いただくためにはファイル更新や「デバイス管理システム」の登録の更新などの手続きが必要となります。
詳細は、担当営業までお問い合わせください。

新機能の提供開始により、ますます便利になるアムニモの通信デバイスとクラウドサービスにつきまして、ご興味のある方は、下記のお問い合わせ先にご連絡ください。
