『Japan IT Week春 2021』出展のお知らせ

この度アムニモは、2021年5月26日(水)から28日(金)※まで東京ビッグサイトの青海展示棟で開催される「Japan IT Week 春2021」の「IoT&5Gソリューション展」に出展します。
なお、アムニモは展示会場での対応人数を最小限に抑え、リモートでの対応も通じて、製品・サービスをご紹介致します。

信頼性のあるシステムで遠隔監視を止めない「IoTデバイス」や、複数のデバイスを効率よく管理・運用できる「クラウドアプリケーション」などを出展します。先月発売されたばかりの「IoTルーター amnimo Rシリーズ AR10」を含む新製品・サービスのデモンストレーションや、現在開発中の屋外版Edge Gatewayや統合ビデオ管理システムなどの参考出展もございます。

▷ 来場をご希望の方はこちらより事前登録をお願い致します。（無料）
※外部サイト（Japan IT Week春）へ遷移します。

※本展示会は4月26日(月)～28日(水)から5月26日(水)～28日(金)へ日時が変更となりました。


※出展イメージ図

【出展製品（IoTデバイス）】

●Edge Gateway amnimo G シリーズ AG10

・４台の監視カメラを接続し、PoEで電力供給可能なIoTゲートウェイ
・高信頼性のSSDを搭載し、監視カメラで撮影した映像をローカル保存可能（最大2TBまで搭載可能）
・電源バックアップにより瞬停、電源ノイズに強い
・通信キャリアの障害発生時に4つのSIMを自動切換え



※Edge Gateway amnimo G シリーズ AG10

●IoTルーター amnimo Rシリーズ AR10

・さまざまな産業用途の機器と接続可能なIoTルーター
・ISDN端末や既存の3G端末の置き換えに最適
・瞬停対策、複数SIM搭載、クラウドアプリケーション連携などの機能を実装
・remot3.it社が提供するリモート接続サービス「remote.it」をプレインストールしており、産業用機器に対するセキュアな遠隔アクセスを実現



※IoTルーター amnimo Rシリーズ AR10

●屋外対応Edge Gateway

※参考出展。開発中

・Edge Gatewayの基本的な機能を継承した屋外モデル
・直射日光や風雨に直接さらされる過酷な場所でも設置可能
・AC100～240V対応、雷対策（SPD）搭載
・4台の監視カメラへのPoEによる電力供給や、撮影した映像の産業用SSDへの保存が可能（最大2TBまで搭載可能）


●AIチップ搭載Edge Gateway

※参考出展。開発中

・Edge GatewayにAIチップを搭載したモデル
・エッジで画像をAI処理。必要データのみクラウド伝送で、通信コスト抑制
・高度な画像認識が可能
・お客様独自のAIモデルも搭載可能


【出展アプリケーション】

●デバイス管理システム

・IoTデバイスの初期設定から設置後の運用までの各種作業をクラウド経由で実施
・保守運用のための人員派遣を最小化
・自動初期設定機能により設定作業を簡略化
・トラブル時も動作ログを遠隔から取得し対応を省力化


●ビデオマネジメントシステム（Nx Witness）

・カメラで撮影した映像の表示、録音、再生、検索を制御
・Edge Gateway amnimo G シリーズ AG10にプレインストールされているためすぐに利用可能
・映像の特定範囲において動きがあったデータをスマート検索で素早くサーチ
　※NxWitnessはNetwork Optix社の製品であり、アムニモ株式会社が代理店として販売いたします


●remote.it

・専用のグローバルIPを持たない機器同士のP2P接続を簡易に実現
・ポートの開放が不要でセキュリティの穴を作らない
・アムニモのIoTデバイスにプレインストールされているため設定作業が簡略
　※remote.itはremot3.it Inc.の製品であり、アムニモ株式会社が代理店として販売いたします


●統合ビデオ管理システム

※参考出展。開発中

・広域に展開するカメラ映像をクラウド上で統一して管理することが可能
・必要なデータのみクラウドにアップロードされるため、通信コスト・回線負担を最小化
・アラーム情報を映像と関連付け、警備のための業務システムとして利用可能
