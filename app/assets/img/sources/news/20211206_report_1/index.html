特許取得のお知らせ「瞬停対策」

この度、アムニモの製品である「Edge Gateway Indoor amnimo GシリーズAG10」と「Edge Gateway Outdoor amnimo GシリーズAG20」の瞬停対策機能に関する特許を取得しましたことをお知らせします。

【特許番号】特許第6954423号
【発明の名称】端末装置
【特許登録日】2021年10月4日

「Edge Gateway Indoor amnimo GシリーズAG10」と「Edge Gateway Outdoor amnimo GシリーズAG20」は、外部からの電源供給が不安定な状況や瞬間的に切断される場合などに備えて、バックアップ電源を実装しています。
外部からの電源の遮断を検知すると、バックアップ電源により装置全体の動作は継続しつつ、PoE によるカメラなどの外部機器への電源供給を停止し、電力の消費量を抑制します。
電源供給を停止すると、カメラなどの外部機器からの情報流入が止まるため、SSDへのデータ書き込みのほとんどが停止され、内部のeMMC へのログの書き込み以外はほとんどのデータ書き込みは行われない状態になります。
この状態で外部からの電源供給が復活しない場合、特にシャットダウンの処理などは行わずに全プロセスが強制遮断になりますが、データの書き込みがほとんど行われないためメモリのクラッシュが発生する可能性は非常に低いです。
今回取得した「特許第6954423号」では、外部電源の遮断を契機として外部機器への電源供給を停止すること、およびその際に外部機器のデータ出力を停止するという点が特許の対象となりました。
